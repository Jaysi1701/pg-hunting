# 🏡 PG Hunting Invite

A fun and interactive website created to invite a friend for a PG hunting mission.

## ✨ Features

- 🌸 Cute animated interface
- ☁️ Moving clouds
- 🏡 Floating PG-themed icons
- 😂 Runaway "No" button
- 🎉 Confetti celebration
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run

1. Clone or download this repository.
2. Open `index.html` in your browser.

Enjoy the experience!
